&nbsp;프로젝트를 수행하면서 필요한 새로운 기술을 빠르게 습득하는 것에 큰 즐거움을 느낍니다. 또한, 한국 문화를 적극 활용하여 일본의 MZ 세대에게 긍정적인 반응을 얻는 가게들을 운영하면서, 고객 중심의 접근과 협력을 중요하게 생각하게 되었습니다. 이러한 경험을 통해 개발의 전반적인 라이프사이클을 이해하고, 팀 프로젝트에서도 효과적으로 기여할 수 있음을 자신합니다.
